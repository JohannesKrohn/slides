Frontify

bit.ly/ux-platform

What can it do?

It is a mixture of confluence, jira and zeplin. It cannot not replace jira or confluence but works well alongside both.

  • You can use it for documentation
  • You can share files
  • You can collaborate, get stakeholder feedback and sign offs

Project space

One of the things that Frontify provides that for example zeplin and figma are missing is a project space where you can also host files. That is convenient for handover and a detailed briefing.

  • Journey Maps
  • Development ready assets
  • Additional or detailed guidances
  • Secondary Studies
  • Deep links to comments

Project views

Each project has a customisable workflow view that follows the kanban board princibles with multiple swim lanes.

  • Inspect Mode
  • Workflow View
  • Project Information view
  • Activity view
